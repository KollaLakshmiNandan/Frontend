CSS Flexbox Interview Questions & Answers

1. What is CSS Flexbox and why is it used?

Answer

Flexbox (Flexible Box Layout) is a layout model in CSS that arranges items in a row or column. It's great for building responsive designs and managing spacing/alignment between items, especially when their size is dynamic.

2. What are the two main axes in Flexbox?

Answer

The two main axes in Flexbox are:

  • Main Axis: The direction items are laid out (defined by flex-direction) property.
  • Cross Axis: Perpendicular to the main axis. Used for vertical/horizontal alignment depending on direction.

3. How do you make an element a flex container?

Answer

You make an element a flex container by setting its display property to either flex or inline-flex.

<div style="display: flex;"> <!-- flex items --> </div>

4. Explain the difference between display: flex; and display: inline-flex;.

Answer
  • display: flex;: makes a container behaves like a block-level flex container.
  • display: inline-flex;: Behaves like a inline element while still using flex layout for its children.

5. What is flex-direction and what are its possible values?

Answer

The flex-direction property establishes the main-axis, thus defining the direction flex items are placed in the flex container.

Possible values:

  • row (default): Arranges flex items horizontally, from left to right
  • row-reverse: Arranges flex items horizontally, but in reverse order, from right to left.
  • column: Arranges flex items vertically, from top to bottom.
  • column-reverse: Arranges flex items vertically, but in reverse order, from bottom to top

    • 6. What is justify-content and how does it work?

    Answer

    Controls horizontal alignment (in row direction) or vertical (in column direction).

    Common values include:

    • flex-start (default): Items are packed towards the start of the main axis.
    • flex-end: Items are packed towards the end of the main axis.
    • center: Items are centered along the main axis.
    • space-between: Items are evenly distributed with the first item at the start and the last item at the end.
    • space-around: Items are evenly distributed with equal space around them.
    • space-evenly: Items are evenly distributed with equal space between them and equal space at the ends.

    7. What is align-items and how does it work?

    Answer

    Aligns items along the cross axis (opposite to main axis).

    Common values include:

    • stretch (default): Items are stretched to fill the container (but still respect min-width/max-width).
    • flex-start: Items are packed towards the start of the cross axis.
    • flex-end: Items are packed towards the end of the cross axis.
    • center: Items are centered along the cross axis.
    • baseline: Items are aligned such that their baselines align.

    8. Explain flex-wrap and its values.

    Answer

    The flex-wrap property Defines whether items should wrap or stay in one line.

    Possible values:

    • nowrap (default): All flex items will be on one line.
    • wrap: Flex items will wrap onto multiple lines from top to bottom.
    • wrap-reverse: Flex items will wrap onto multiple lines from bottom to top.

    Use it when flex items should move to the next line if needed.

    9. What is the shorthand property for flex-direction and flex-wrap?

    Answer

    The shorthand property is flex-flow.

    <div style="display: flex; flex-flow: row wrap;"> <!-- flex items --> </div>

    10. What is align-content and when is it used?

    Answer

    The align-content property is used to Aligns multiple flex lines. only works with flex-wrap: wrap;. or flex-wrap: wrap-reverse;

    Common values include flex-start, flex-end, center, space-between, space-around, stretch (default).

    11. Explain the flex shorthand property for flex items.

    Answer

    The flex shorthand property sets the flex-grow, flex-shrink, and flex-basis properties on a flex item.

    .flex-item { flex: <flex-grow> <flex-shrink> <flex-basis>; }
    • flex-grow: Defines the ability for a flex item to grow if necessary (default is 0).
    • flex-shrink: Defines the ability for a flex item to shrink if necessary (default is 1).
    • flex-basis: Defines the default size of an element before any free space is distributed (default is auto).

    Examples:

    • flex: 1; is equivalent to flex: 1 1 0%; (grow, shrink, and take up 0% basis, but grow to fill available space).
    • flex: auto; is equivalent to flex: 1 1 auto; (grow, shrink, and take up content size).
    • flex: none; is equivalent to flex: 0 0 auto; (do not grow, do not shrink, and take up content size).

    12. What is flex-grow and when would you use it?

    Answer

    flex-grow Defines how much a flex item can grow relative to others.

    Examples:

    • flex-grow: 2; takes double the space compared to flex-grow: 1;

      • It's used when you want certain items to expand and fill available space.

    13. What is flex-shrink and when is it useful?

    Answer

    The flex-shrink property Controls how much an item shrinks when space is tight. A value of 1 means it will shrink equally with other items. A value of 0 prevents the item from shrinking.

    It's useful when you need to control how items behave when the container gets smaller and there isn't enough room for all items at their preferred sizes.

    14. What is flex-basis?

    Answer

    The flex-basis property Sets the initial size of a flex item before remaining space is distributed. Can be auto, 0, 200px, etc.

    15. How do you center an item both horizontally and vertically using Flexbox?

    Answer

    To center an item both horizontally and vertically, you apply justify-content: center; and align-items: center; to the flex container.

    .flex-container { display: flex; justify-content: center; /* align-items: center; /*}

    Also set a height for vertical centering (e.g., 100vh).

    16. What is the order property and how does it work?

    Answer

    The order property controls Controls the visual order of flex items (does not affect HTML).

    The default value for order is 0. Negative values are allowed.

    /*.item-1 { order: 2; } */

    17. What is align-self and when would you use it?

    Answer

    The align-self property allows individual flex items to override the align-items for a single item. Useful when one item needs different vertical alignment.

    It accepts the same values as align-items (auto (default), flex-start, flex-end, center, baseline, stretch).

    It's useful when you have a general alignment for most items but need to adjust one or a few items differently.

    18. Can a flex item also be a flex container? Provide an example.

    Answer

    Yes! You can nest flex containers. A flex item can itself use display: flex; to control its children.

    19. What are the advantages of using Flexbox over traditional layout methods (floats, positioning)?

    Answer

    Advantages of Flexbox include:

    • Simpler, cleaner layout code
    • Easy alignment and spacing
    • Built-in responsiveness
    • No need for float or clearfix hacks
    • Visual order control with order

    20. When would you choose CSS Grid over Flexbox, and vice-versa?

    Answer
    • Choose Flexbox when: You need to lay out items in a single dimension (either a row or a column). It's great for components where items need to be aligned and distributed along one axis, like navigation bars, forms, or a series of cards.
    • Choose CSS Grid when: You need to lay out items in two dimensions (rows and columns) simultaneously.

    Often, they are used together: Grid for layout, and Flexbox for components.
