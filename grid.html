⇒using grid:

  • CSS Grid Layout is a two-dimensional system — it lets you design web layouts using both rows and columns.
  • Grid containers are defined using display: grid or display: inline-grid, and all direct children become grid items.
  • You define the structure using grid-template-rows and grid-template-columns, allowing precise control over layout areas.
  • Grid supports alignment, spacing (gap), and item positioning (grid-row, grid-column), enabling complex responsive designs with clean code.

⇒using flex:

  • Flexbox is a one-dimensional layout system — it arranges items in a row or column (not both like Grid).
  • Use display: flex or inline-flex on a container — its direct children become flex items.
  • The flex-direction property controls the main axis — e.g., row, column, row-reverse, or column-reverse.
  • Flexbox allows easy alignment and spacing — with properties like justify-content, align-items, gap, and flex-grow.
