HTML & CSS Interview Questions

HTML Questions

1. What does HTML stand for?

Answer

HTML means HyperText Markup Language. It's the base language used to build web pages. It tells browsers how to display headings, paragraphs, links, images, and forms. Without HTML, websites can't be structured.

2. What is the basic structure of an HTML page?

Answer

A basic HTML page starts with <!DOCTYPE html> to define the document type. Then it has <html>, inside which there is a <head> for meta info like title and CSS, and a <body> for the visible content. This is the skeleton of any web page.

3. What is the difference between an element and a tag?

Answer

A tag is the actual code like <p> or </p>. An element includes the opening tag, content, and closing tag — for example, <p>Hello </p> is a paragraph element.

4. What are attributes in HTML?

Answer

Attributes give extra info about elements. They go inside the opening tag. For example, in <img src="pic.jpg">, the src attribute tells the browser where to find the image.

5. What are void elements?

Answer

Void elements are self-closing — they don’t have closing tags. Examples: <br>, <img>, <hr>. They just do their job and end right there.

6. How do you create a hyperlink?

Answer

Use the <a> tag with an href attribute. Example: <a href="https://google.com">Click Here </a>. It makes the text clickable and opens the link.

7. What is the purpose of the alt attribute on an <img> tag?

Answer

alt gives a text description for an image. It's shown if the image doesn't load and helps screen readers for accessibility.

8. Differentiate between block-level and inline elements.

Answer

Block elements (like <div>, <p>) take full width and start on a new line.

Inline elements (like <span>, <a>) fit inside a line and don't break flow.

9. What are the three main types of lists in HTML?

Answer

1. Unordered List (<ul>): A list of items marked with bullets.
2. Ordered List (<ol>): A list of items marked with numbers or letters.
3. Definition List (<dl>): A list of terms (<dt>) and their descriptions (<dd>).

Each has different use cases depending on how you want to show data.

10. How do you add comments in HTML?

Answer

Comments in HTML start with <!-- and end with -->. They ignored by the browser.

Example: <!-- This is a comment -->

11. What is the difference between <strong> and <b> tags?

Answer

Both make text bold, but <strong> semantic meaning — it says the text is important. <b> just makes text bold without any special meaning.

12. What are semantic HTML elements?

Answer

Semantic elements clearly explain what the content is. Tags like <header>, <footer>, <section>, and <article>

make the code more readable for developers and search engines.

13. How do you create a simple table?

Answer

Use the <table> tag. Inside it, use <tr> for rows, <th> for headings, and <td> for cells.

14. What is an <iframe> used for?

Answer

<iframe> is used to embed another webpage inside your page. For example, you can embed a YouTube video or a Google Map using it.

15. What are HTML entities?

Answer

HTML entities are used to display special characters like <, >, &, etc., that can't be written directly. Example: & lt; shows <, & copy; shows ©.

16. What does the <!DOCTYPE html> declaration do?

Answer

It tells the browser, “Hey! This page is written in HTML5.” It helps the browser render the page correctly.

17. What is the purpose of the <head> element?

Answer

<head> stores info about the page — like the title, character set, CSS links, and meta tags. It doesn't show anything on the screen.

18. How do you create a form in HTML?

Answer

You use the <form> tag. Inside the form, you can use various input elements like <input type="text">, <input type="radio">, <input type="submit">, <textarea>, etc.

19. What is the purpose of the <label> tag?

Answer

<label> is used to name input fields, making forms more accessible. When you click the label, it focuses the related input box.

20. What is the difference between <div> and <span>?

Answer

<div is a block-level <div> is a block-level element used for layout and containers. <span> is inline, used to style or group small pieces of text.

21. How can you open a link in a new tab?

Answer

By using the target="_blank" attribute in the anchor tag: <a href="url" target="_blank">Link</a>.

22. How do you add a video to a webpage?

Answer

With the HTML5 <video> tag. You can specify the video source using the src attribute or a <source> element inside the video tag. Example: <video src="movie.mp4" controls></video>.

23. What does the colspan attribute do in a table?

Answer

The colspan attribute on a <th> or <td> to make one cell spread across multiple columns.

24. What is the lang attribute in the <html> tag for?

Answer

The lang attribute specifies the language of the document's content. This helps search engines and browsers. For example, lang="en" for English.

25. Can you have multiple <h1> tags on a single page?

Answer

Yes, but only if they're inside different sections or articles.For SEO, it's better to have one main <h1> for the page title and use <h2>, <h3>, etc. for subheadings.

CSS Questions

26. What does CSS stand for?

Answer

CSS stands for Cascading Style Sheets. It's used to describe the presentation and styling of a document written in a markup language like HTML.

27. What are the three ways to include CSS in a webpage?

Answer

1. External CSS: A separate .css file linked via the <link> tag in the <head>. (Best practice)
2. Internal CSS: Placed inside a <style> tag within the <head> section.
3. Inline CSS: Applied directly to an element using the style attribute.

28. What is a CSS selector?

Answer

A selector is the part of a CSS rule that "selects" the HTML element(s) you want to style. Examples include selecting by element type (p), class (.my-class), or ID (#my-id).

29. What is the difference between an ID and a class selector?

Answer

An ID (#my-id) is unique and should only be applied to a single element on a page. A class (.my-class) can be applied to multiple elements.

30. What is the CSS Box Model?

Answer

The CSS box model describes the rectangular boxes that are generated for elements in the document tree. Each box has four edges: the margin edge, border edge, padding edge, and content edge.

31. What is the difference between margin and padding?

Answer

Padding is the space between the element's content and its border. Margin is the space outside the border, between the element and other elements.

32. How do you center a block-level element horizontally?

Answer

You can center a block-level element by setting its width and then setting its horizontal margins to auto. Example: width: 80%; margin: 0 auto;

33. What is the display property in CSS?

Answer

The display property specifies how an element is shown on a page. Common values are block, inline, inline-block, and none.

34. What's the difference between visibility: hidden; and display: none;?

Answer

display: none; removes the element completely from the document flow. It takes up no space.

visibility: hidden; hides the element, but it still occupies its space in the layout.

35. What are pseudo-classes?

Answer

A pseudo-class is used to define a special state of an element. For example, :hover applies a style when the user hovers over an element, :active when it is clicked, and :focus when it is selected.

36. What is specificity in CSS?

Answer

Specificity is the algorithm used by browsers to determine which CSS rule is the most "specific" and should be applied to an element when there are conflicting rules. IDs are more specific than classes, which are more specific than element tags.

37. What is the z-index property used for?

Answer

The z-index property specifies the stack order of positioned elements (elements whose position is not static). An element with a higher z-index will appear in front of an element with a lower one.

38. What are the possible values for the position property?

Answer

static: Default value. Elements render in order, as they appear in the document flow.
relative: The element is positioned relative to its normal position.
fixed: The element is positioned relative to the browser window and does not move when scrolling.
absolute: The element is positioned relative to its nearest positioned ancestor.

39. How do you change the color of text?

Answer

Using the color property. Example: p { color: blue; } or p { color: #0000FF; }.

40. How do you set a background image?

Answer

Using the background-image property. Example: body { background-image: url("paper.gif"); }.

41. What is the difference between em and rem units?

Answer

em is relative to the font-size of its direct or nearest parent element. rem (root em) is relative only to the font-size of the root <html> element.

42. How would you apply a style to every p element that is a direct child of a div?

Answer

Using the child combinator (>): div > p { ... }.

43. What does the float property do?

Answer

The float property specifies whether an element should float to the left or right, allowing text and inline elements to wrap around it. It essentially takes the element out of the normal document flow.

44. How can you remove the underline from a hyperlink?

Answer

By setting the text-decoration property to none. Example: a { text-decoration: none; }.

45. What is the purpose of the universal selector (*)?

Answer

The universal selector (*) selects all HTML elements on the page. It's often used in CSS resets to remove default browser margins and paddings.

46. How do you make text bold in CSS?

Answer

Using the font-weight property. Example: p { font-weight: bold; } or p { font-weight: 700; }.

47. What does the box-sizing: border-box; property do?

Answer

By default, an element's width and height properties apply to the content box. box-sizing: border-box; tells the browser to include the padding and border in the element's total width and height, which makes layout calculations more predictable.

48. Can you combine multiple selectors?

Answer

Yes, you can group selectors to apply the same styles to multiple elements by separating them with a comma. Example: h1, h2, p { color: navy; }.

49. How do you specify a comment in a CSS file?

Answer

CSS comments start with /* and end with */. Example: /* This is a CSS comment */.

50. What does the opacity property do?

Answer

The opacity property sets the opacity level for an element. The value ranges from 0.0 (fully transparent) to 1.0 (fully opaque). It affects the entire element, including its content.
